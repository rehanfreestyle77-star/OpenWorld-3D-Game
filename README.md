## 🚀 Getting Started (Run from Source)

### Prerequisites
- Unity Hub + Unity Editor (2021.3 LTS or higher recommended)
- Android Build Support module installed

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/rehanfreestyle77-star/OpenWorld-3D-Game.git
   ```

2. **Open in Unity**
   - Open Unity Hub → **Add project from disk**
   - Select the cloned folder

3. **Build for Android**
   - `File → Build Settings → Android`
   - Click **Switch Platform**
   - Click **Build** → select output folder

> Or just **[download the APK directly](https://rehanfreestyle77.itch.io/zombie-slayer)** and install on your Android device!
